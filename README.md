## 🌍 Problem Statement & UN SDG Alignment
Small scale businesses operate with slim margins and struggle with clean ledger reviews. Hiring standard management consulting agencies is too expensive. 

This standalone Python tool bridges the gap by translating standard transactional logs into clear financial actions. This project aligns directly with **UN SDG Goal 8: Decent Work and Economic Growth** by protecting small local businesses against operational bankruptcy.

---

## ✨ Features
- **File Ingestion System:** Supports any industry standard CSV data log files.
- **Dynamic Graphics UI:** Interactive Plotly Express charts adjustable on custom axes.
- **Automated AI CFO Engine:** Uses Google Gemini 2.5 Flash custom rulesets to isolate structural leakages.
